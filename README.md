## Начиная

Сначала запустите сервер разработки:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Откройте [http://localhost:3000](http://localhost:3000) с помощью браузера, чтобы увидеть результат.
